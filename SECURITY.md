# Security Policy

## Reporting a vulnerability

Please report security issues responsibly.

Preferred: GitHub Security Advisories (if enabled for this repository).
If Advisories are not available, open a public issue with sensitive
information redacted.

## What not to share

Do not disclose:
- Wi-Fi credentials
- tokens or API keys
- private keys or certificates
- personally identifiable information

## What to include

Provide a minimal reproduction description, the device/firmware version,
and the impact observed. Keep reports concise and focused on the security
issue.
