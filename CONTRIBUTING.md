# Contributing

Thank you for your interest. This repository is in a public read-only
phase.

## External contributions

External PRs are not accepted yet. They are reserved for post-M8
collaboration.

## Issues

Issues are enabled for bug reports only (not feature requests). Maintainers
may not respond, and there is no support obligation.

If you submit a bug report, include:
- device model and hardware revision
- firmware version
- steps to reproduce (minimal, exact)
- expected vs actual behavior

Do not post secrets. Redact Wi-Fi credentials, tokens, private keys, or any
other sensitive information.
